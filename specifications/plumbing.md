# Plumbing Specification

All plumbing work to comply with Mexican plumbing codes.

**Project Scope:** Existing plumbing locations will be utilized. New plumbing locations are not possible. Cabinet layout must accommodate existing plumbing infrastructure.

---

## Water Supply (EXISTING LOCATIONS)

### Sink
- **Location:** Continuous Base Cabinetry
- **Type:** To be specified
- **Supply:** Hot and cold water (existing connections)
- **Requirements:** 
  - Use existing supply lines
  - Verify shut-off valves are functional and accessible
  - Cabinet layout must accommodate existing plumbing location

### Dishwasher
- **Location:** Continuous Base Cabinetry (adjacent to sink)
- **Supply:** Hot water connection (existing or shared with sink)
- **Requirements:** 
  - Use existing connection if present
  - If no existing dishwasher connection, tap from sink supply
  - Shut-off valve accessible

### Refrigerator (if ice maker/water dispenser)
- **Location:** Tall Cabinetry Wall - 908mm Refrigerator Section
- **Supply:** Cold water line (if existing connection available)
- **Requirements:** 
  - Only if existing water line is present in this location
  - If no existing line, refrigerator model without water/ice features
  - Shut-off valve accessible if used

### Island
- **Location:** Kitchen Island
- **Status:** No plumbing available
- **Note:** Island will not have sink or water features

---

## Drainage (EXISTING LOCATIONS)

### Sink
- **Location:** Continuous Base Cabinetry
- **Requirements:** 
  - Use existing drain location
  - Verify P-trap and venting are functional
  - Cabinet layout must accommodate existing drain location

### Dishwasher
- **Location:** Continuous Base Cabinetry
- **Requirements:** 
  - Drain connection to sink drain or existing dedicated drain line
  - Verify connection is functional

---

## Pre-Installation Requirements

1. **Survey existing plumbing** - Map all current supply and drain locations
2. **Test all connections** - Verify supply lines, shut-offs, and drains are functional
3. **Plan cabinet layout** - Ensure cabinet design accommodates fixed plumbing locations
4. **Verify accessibility** - All shut-off valves must remain accessible after cabinet installation

## Environmental Considerations
All materials and connections suitable for humid tropical conditions.
